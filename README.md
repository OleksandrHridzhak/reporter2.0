# Reporter 2.0 – Генератор Звітів ДСТУ + ЛНУ

Веб-застосунок для швидкого створення академічних звітів відповідно до стандартів **ДСТУ** та вимог **ЛНУ ім. Івана Франка**, з інтеграцією **AI-асистента Gemini**.

## Можливості

- 📄 **Заготовлені блоки**: Титульна сторінка, Анотація/Мета, Хід роботи, Висновки, Список джерел
- 🎨 **Стандартне форматування**: Поля відповідно до ДСТУ (ліво 30мм, право 15мм, верх/низ 25мм), шрифт Times New Roman 14pt, міжрядковий інтервал 1.5
- �� **AI-асистент (Gemini)**: Ліва панель чату з Gemini 1.5 Flash для генерації тексту по кожному блоку
- ✏️ **Ручне редагування**: Всі поля редагуються вручну
- ⬇️ **Експорт в DOCX**: Готовий файл з правильним форматуванням
- 💾 **Автозбереження**: Чернетка зберігається в localStorage
- 📂 **Збереження/завантаження JSON**: Можна зберегти звіт як JSON і продовжити пізніше

## Запуск

```bash
# Встановити залежності
npm install

# Запустити в режимі розробки
npm run dev

# Зібрати для продакшену
npm run build
```

Відкрийте http://localhost:5173 у браузері.

## Використання Gemini AI

1. Отримайте **безкоштовний** API ключ на [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Вставте ключ у поле «API Ключ Gemini» в лівій панелі
3. Виберіть **цільовий блок** зі списку (або клікніть на блок у редакторі)
4. Введіть запит і натисніть **▶ Надіслати** (або Ctrl+Enter)
5. Натисніть **↩ Вставити** щоб застосувати відповідь до блоку

## Структура звіту

| Блок | Вміст |
|------|-------|
| 📄 Титульна сторінка | Університет, факультет, кафедра, тема, студент, викладач |
| 📋 Анотація / Мета | Мета роботи, перелік завдань, засоби виконання |
| 🔧 Хід роботи | Кроки виконання (необмежена кількість) |
| ✅ Висновки | Підсумки роботи |
| 📚 Список джерел | Бібліографічний список (нумерований) |

## Технічний стек

- **React 19** + **TypeScript** + **Vite**
- **@google/generative-ai** – Gemini API
- **docx** – генерація DOCX файлів
- **file-saver** – збереження файлів
